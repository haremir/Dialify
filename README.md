# Dialify

Metin belgesindeki telefon numaralarını temizleyip `phone` sütunlu CSV dosyasına çeviren basit Streamlit uygulaması.

## Özellikler

- `.txt` ve `.csv` dosyası yükleme
- Telefon numaralarını `+90` formatına çevirme
- Tekrar eden numaraları temizleme
- `phone` sütunlu CSV indirme

## Çalıştırma

```bash
pip install -r requirements.txt
streamlit run app.py
```
